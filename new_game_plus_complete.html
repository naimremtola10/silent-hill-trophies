🏆 Puppetmaster (Silver) - ENDING 2 BOSS
Boss: Tsukomogami

Trophy Requirement: Get through arrow phase WITHOUT being hit

Arrow Phase:
• 3 arrows in air → dodge when close (3 dodges)
• Volley of quick arrows → multiple fast dodges
• Arrow rain → dodge constantly
• Run forward to boss to end phase

Strategy:
• Equip Suzuran + Boar Omamoris (Perfect Dodge + Stamina)
• OR Shrew Mole (dodges don't cost stamina)
• Waves can hit you (doesn't void trophy)
• Only ARROWS void trophy!

💾 Make save before boss!

First fight Shiromuku briefly, then Shichibi appears.

Trophy Requirement: Kill ALL adds before damaging boss

Strategy:
• Boss weakens 3 times → spawns enemies
• Kill every enemy during each phase
• THEN attack boss
